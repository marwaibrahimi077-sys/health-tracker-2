

# Student Health & Wellness Tracker

## Description
This is a web-based Student Health & Wellness Tracker designed to help students monitor and improve their daily habits, including sleep, hydration, exercise, screen time, and skin care. The website provides charts and tips to support healthy routines.

## Features
- Track daily habits and wellness activities
- Interactive charts that update in real time
- Export data button to save personal progress
- Friendly health tips and guidance
- Fully client-side: all data stays in the browser

## How to Use
1. Open `index.html` in a browser or visit the GitHub Pages link.
2. Fill out the tracker forms.
3. View your charts and monitor progress.
4. Use the "Export Data" button to save your entries.
